[No.1] 危険警告型

「利回り10%超」で案内される新築木造アパート。
まじで危ない。

理由は単純：
新築プレミアム（新しさの価値）が
5年で8割消える＆
木造は法定耐用年数が22年。

つまり築5年で既に売値が3割下がってる。
利回りは実質6%台に化けてる。

不動産会社は「新築時点」の利回りしか見せない。
出口戦略を持たずに買うな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

金融機関の融資判定が「物件の価値」じゃなく
「あなたの返済力」で決まることを
知ってる投資家は少ない。

築古高利回り物件が融資つかないのは
「儲かりそう」だからじゃなく
「返済できなくなる可能性」を
銀行が計算してるからだよ。

年収と勤続年数。
これが全てを決める。

推奨投稿時間：朝8時 / 昼12時 / 夜20時のいずれか

[No.3] 失敗談型

知人の新築一棟木造投資。
建築会社が倒産して施工不良が発覚。
契約書に「完工後の瑕疵（かし）責任」が
1年間だけ。

1年2ヶ月後、雨漏り発生。
自腹で修繕費800万。

建築詐欺とまでいかずとも
「契約書の条件」で人生変わる。
必ず弁護士のチェック入れろ。

[No.4] 初心者罠型

投資初心者が犯す最大の誤りは
「表面利回り」と「実質利回り」の区別がない。

表面利回り＝年間家賃÷購入価格
実質利回り＝（年間家賃−経費）÷購入価格

経費には：管理費、修繕積立、固都税、火災保険…

この差が5%以上あることもザラ。
電卓だけで判断するな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.5] 融資・ファイナンス型

融資を断られる人の共通点：

①勤続年数2年未満
②転職歴が多い
③自己資金が物件価格の10%未満
④クレカの遅延履歴あり
⑤年収に対して既存ローンが大きい

銀行は物件じゃなく「あなたの信用」を査定する。
年収500万で既存ローン4000万は99%落ちる。

推奨投稿時間：朝8時 / 昼11時 / 夜19時のいずれか

[No.6] 利回りの罠型

「利回り8%」という数字の罠。

買値3000万、年間家賃240万＝8%

でも現実：
・空室率5%→228万
・管理費月2万→24万
・修繕費年50万→50万
・固都税年30万→30万
・火災保険年8万→8万

実質：228−124=104万÷3000万＝3.5%

数字は嘘つかないが、選べる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.7] 市場データ型

2024年の首都圏新築分譲マンション
平均価格：6390万円（前年比+8.2%）

なぜか。

金利上昇で購買層が絞られ、
高年収層だけが買える＆
建築費インフレで原価が上がってるから。

年収1000万以下で
新築マンション買うのは
「割高掴み」のリスク高い。

[No.8] 売却タイミング型

「いつ売るか」で利益は180度変わる。

築5年：新築プレミアム8割消滅
築10年：修繕のピークが見えてくる
築15年：金融機関の評価が急落

売り時は築3〜5年。

その後は修繕費が雪だるま式に増える。
「ずっと保有」は幻想だよ。

推奨投稿時間：朝10時 / 昼15時 / 夜23時のいずれか

[No.9] 自宅購入型

自宅購入で誰も教えてくれないこと：

建物は資産じゃなく「負債」。
毎年減価償却する（価値が落ちる）。

土地だけが資産。

だから「駅近で利便性高い土地」に
シンプルな建物を建てるのが
20年後に生き残る家選びだよ。

推奨投稿時間：朝9時 / 昼14時 / 夜21時のいずれか

[No.10] 管理・運用型

物件買ってから始まる本当の苦労：

入居者クレーム、設備故障、
家賃滞納、原状回復トラブル。

不動産投資家は
実は「運営者」であって投資家じゃない。

これを管理会社に丸投げすると
手数料5%+修繕費で利回りが3%に。

自分でやるか、それを前提に買うか。
最初の判断で全て決まる。

推奨投稿時間：朝10時 / 昼13時 / 夜22時のいずれか

[No.11] 相続・贈与型

不動産と相続で揉える家族の典型：

複数の兄弟が相続人。
親が不動産を残した。

「売るか」「誰が取るか」で対立。
売るにも全員の同意が必要。

固都税は毎年かかる。

相続税対策で不動産買う人多いけど
「その後の兄弟間争い」は計算に入ってない。
弁護士と税理士、両方呼べ。

推奨投稿時間：朝8時 / 昼11時 / 夜20時のいずれか

[No.12] エリア選定型

なぜ駅から徒歩10分と15分で
価格差が30%になるか。

駅近は「需要と供給」のゲーム。
駅から遠いと「若年層」が消える。

投資判断：
若年層が集まるエリア→賃借人が途切れない
オフィス街→24時間需要→安定
郊外ニュータウン→30年後は廃墟の可能性

立地選ぶ時点で
30年後の価値が80%決まってる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.13] 金利・ローン型

「変動金利0.5%」という誘い文句。

危険だよ。

変動金利は6ヶ月ごとに見直し。
日銀が利上げ局面なら
3年後に2%になってる可能性あり。

年間返済額が200万→300万に跳ね上がる。

固定金利1.5%で15年か20年。
「安さ」より「予測可能性」を優先。

推奨投稿時間：朝8時 / 昼12時 / 夜19時のいずれか
