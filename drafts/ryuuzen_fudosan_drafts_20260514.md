[No.1] 危険警告型

新築木造アパート、
「利回り8%、税金対策に最高」
とか言われてるやつ。

まじで危ないからな。

木造は17年で償却。
その後、修繕費が跳ね上がる。
金融機関は築20年を境に
一気に評価を落とす。

税理士が「節税になります」は
その年だけの話。
10年後の出口戦略も描いてから買え。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

不動産会社は
「売却時期」は絶対に言わない。

売ったら仲介手数料が発生するから
むしろ買わせ続けたいんだよな。

10年持ったら売却も視野に入れるべき。
でも売却タイミングを相談すると
「まだ保有してほうが…」って返ってくる。

これが業界の本音。
自分で相場観を持つしかない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.3] 失敗談型

知人が2億の新築物件を買った。

「建築費が安い」って

完成後、雨漏り・基礎ヒビが連発。
修繕費だけで500万超。
その施工会社、もう存在しない。

建築詐欺じゃなくても
「安すぎる」は赤信号。
複数の建築士に見積もり取れ。

[No.4] 初心者罠型

投資初心者の大半が
「利回り」だけで物件を選ぶ。

でも実際に必要なのは
・空室リスク
・修繕積立金
・管理会社の質
・出口戦略

この4つ見落とすと
高利回りに見えても、赤字物件になる。

見た目の数字より
「実際に自分が運用できるか」を基準に。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.5] 融資・ファイナンス型

融資が断られる奴の共通点。

①自営業なのに決算書がめちゃくちゃ
②クレジットカードの滞納履歴がある
③既存ローンの返済が甘い

金融機関は数字を見る。
「年収1000万です」じゃなくて
「税務申告書で確認できる所得」が全て。

自分のファイナンス状況を知らない投資家多すぎ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.6] 利回りの罠型

表面利回り8%の物件。

でも実際は
・管理費2%
・修繕費1.5%
・空室率15%

実質利回りは3.5%。

これなら銀行預金のほうがマシ。

「表面利回り」を言ってくる営業は
信用しちゃダメ。
自分で実質利回りを計算してから判断しろ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.7] 市場データ型

首都圏の中古一棟木造アパート。

2年前：3,000万
現在：2,500万

理由は単純。
金利上昇 + 修繕懸念。

逆に駅5分以内の
鉄骨造・新耐震は
むしろ価格据え置き or 上昇。

「木造だから安い」は罠。
構造と立地で価値が全く変わる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

「今が売り時か買い時か」の判断軸。

売り時：
・金利上昇局面（今ここ）
・築15年以上の木造
・周辺に競合物件が増えている

買い時：
・金利が安定している
・築10年以下の鉄骨
・駅徒歩圏の利便性が高い

タイミング感が投資の7割。
焦って買うな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.9] 自宅購入型

家を買うときに誰も教えてくれないこと。

「月々の返済額」だけ見てる奴多い。

でも固定資産税・火災保険・修繕費も
毎年かかる。

さらに30年後、外壁塗装だけで
200万超の出費がある。

「購入金額」じゃなく
「生涯所有コスト」で判断しろ。

テナント対応・クレーム処理・
修繕交渉・空室募集・滞納回収…

不動産会社に丸投げしても

自分で年2回は現地確認。
テナントの顔も知っておく。

これやらない投資家は
後々後悔する。

[No.11] 相続・贈与型

不動産×相続で揉める家族のパターン。

「父が遺した建物、兄弟で分割したい」

でも建物は分割できない。
だから片方が買い取ると相続税が爆発する。

事前に「誰が相続するか」を決めておく。
税理士に相談する。
遺言書を残す。

これやらずに相続を迎えると
兄弟が絶縁する。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.12] エリア選定型

なぜそのエリアは上がるのか・下がるのか。

下がるエリア：
・駅徒歩15分以上
・近隣に空き家が増えている
・人口流出地域

上がるエリア：
・大型開発予定（駅新設など）
・若い世帯の転入が続いている
・交通利便性が急改善

数字で見える理由がない買い物は
投機になる。
統計データを見ろ。
