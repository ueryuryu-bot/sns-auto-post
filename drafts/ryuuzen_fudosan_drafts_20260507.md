[No.1] 危険警告型

新築木造アパートを買う奴、まじで危ねえ。

利回り8%で飛びつくんだけど、実際は
・15年で大規模修繕200万単位
・木造は融資が短期しか出ない
・売却時に価値が半分以下

俺の知人は「節税になる」って理由で買ったら、
10年後に修繕費＋売却損で1000万溶かしてた。

新築は業者の利益が乗ってるんだよな。

[No.2] 業界本音型

不動産会社は「利回り」しか言わない。

なぜなら、言うと物件が売れなくなるから。

本当は語るべきこと：
・空室発生時の2ヶ月間の無収入
・退去者の原状回復費（20万は最低）
・満室でも手取りは表面の60%程度

結局、売主の手数料がゴールだから
「こんなリスクもあります」なんて言わない。

投資家目線の営業は、実は稀だったりする。

推奨投稿時間：朝8時 / 昼12時 / 夜20時

[No.3] 失敗談型

建築詐欺で1500万消えた時の話。

工事が始まったら、施工業者が夜逃げ。
資金を前払いしてたから、金は戻らず。

融資を出した銀行も「施工会社の責任」で終わり。
結局、俺が数百万かけて別業者に修繕させた。

その時気づいたのは「実績」と「資本金」を
事前に調べることの大切さ。

今は請負契約で必ず
保証人と担保を確認してる。

[No.4] 初心者罠型

投資初心者が買う1軒目、大体失敗してる。

なぜなら「満室前提で利回り計算」しちゃうから。

現実：
・新規物件は最初3ヶ月は50%空室が当たり前
・想定賃料＜実際の賃料（新規物件は安く埋める）
・想定利回り9%→実質5%みたいなことばかり

1軒目は利回りじゃなく「立地」と「管理会社」で選べ。

回収は後からついてくるんだよな。

[No.5] 融資・ファイナンス型

融資が通らない奴、大体パターンが同じ。

・自営業だけど決算書が汚い
・年収は高いが変動が大きい
・既に4件以上ローンがある

銀行は「返済余力」を見る。
年収1000万でも負債が800万あると、
「返せる額」は限定的だったりする。

融資を甘く見てる奴が一番危ない。

事前に金融機関に相談しろ、まじで。

[No.6] 利回りの罠型

「表面利回り9%」と「実質利回り4.5%」。

これ、同じ物件だったりする。

表面利回り＝年間家賃÷物件価格だけ。

実質利回りは
（年間家賃－全経費）÷物件価格。

全経費ってのは：
・管理費
・固定資産税
・火災保険
・空室ロス
・修繕積立

これ全部引くと、
見た目の利回りが半分になるんだよな。

推奨投稿時間：朝8時 / 昼13時 / 夜21時

[No.7] 市場データ型

2025年1月時点での相場。

首都圏新築ワンルーム：1000万〜1300万
（想定利回り4.5〜5.5%）

中古ファミリー物件：2000万前後
（立地次第で利回り5〜7%）

地方中古アパート：500〜800万
（表面利回り8〜12%だが空室が課題）

金利上昇で新築の利回りは
全体的に硬直してる。

掘り出し物は中古か地方。
そこに目がいかない投資家が多い理由は「営業がいないから」だったりする。

推奨投稿時間：朝10時 / 昼12時 / 夜19時

[No.8] 売却タイミング型

「今が売り時か、買い時か」の判断軸。

買い時：
・金利が下がり始めた時
・市場に供給が少ない時期
・自分の融資余力がある時

売り時：
・購入時より相場が30%以上上がった時
・築15年超で修繕が迫ってる時
・融資が10年以内に終わる物件

「ずっと持つ」が正解じゃない。
タイミングを誤ると修繕費が利益を喰う。

15年のスパンで判断しろ。

推奨投稿時間：朝9時 / 昼14時 / 夜20時

[No.9] 自宅購入型

自宅を買うとき、誰も教えてくれないこと。

「ローン年数が長いほど、
トータルの返済額が膨らむ」

35年ローンで3000万借りると、
金利2%でも総返済額は3800万超。

つまり800万を銀行に献上してる。

さらに：
・固定資産税は毎年
・修繕費も自己負担
・売却時に手数料3%以上

「持ち家は資産」じゃなく、
「負債＋毎年のコスト」だったりする。

推奨投稿時間：朝8時 / 昼15時 / 夜21時

[No.10] 管理・運用型

物件を買った瞬間から、本当の苦労が始まる。

・クレーム対応（夜中の水漏れとか）
・退去時の原状回復交渉
・税務申告の書類作成
・空室が埋まるまでの営業活動

「運用は管理会社に任せる」って言っても、
最終判断は投資家。

管理会社のレベルで
利回りが1〜2%変わることもある。

相手を見誤ると、3年で数百万溶ける。

物件選び以上に「パートナー選び」が大事なんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜20時

[No.11] 相続・贈与型

不動産で家族が揉める典型パターン。

親が複数の物件を保有したまま相続。

・評価額が不動産会社で100万差がつく
・兄弟で「誰が管理するか」で対立
・空室物件の赤字を巡って泥沼化

相続後に売ろうとしても、
共有名義だと1人が売却を拒否すると動けない。

相続前に「整理して売却する」か
「誰が継ぐか」を決めておくべき。

不動産が「相続問題の火種」になるのは、
決断を先送りにした結果だったりする。

推奨投稿時間：朝8時 / 昼11時 / 夜19時

[No.12] エリア選定型

「なぜそのエリアは上がるのか」

駅徒歩10分以内か、駅から遠いか。

首都圏で「駅まで15分」は
相場で15〜20%マイナス。

さらに：
・学校が近い → ファミリー需要で安定
・商業施設がない → 投資用途が限定
・人口減地域 → 今後も下がる

つまり「物件選び」じゃなく「街選び」が先。

この視点を持ってる投資家は実は少ない。

[No.13] 金利・ローン型

住宅ローンと投資用ローンは全く別。

住宅ローン：
・金利が低い（1.5〜2.5%）
・返済期間が長い（35年）
・審査が厳しい

投資用ローン：
・金利が高い（3〜4.5%）
・返済期間が短い（15〜20年）
・利息控除で節税できる

「投資用」で借りた方が、
トータルでは得だったりする。

その理由は「利息が経費扱い」だから。

金融機関の営業は都合のいいことしか言わないんだよな。

推奨投稿時間：朝9時 / 昼14時 / 夜20時

[No.14] 問いかけ型

「不動産を持つことが、本当に資産形成か」

年間120万。

これ、給与で月5万稼ぐより難しくないか。

時間もかかるし。

「資産」と「手間」のバランスを
誰も教えてくれない。

2024年から変わった制度で、
知らないと損する奴多い。
