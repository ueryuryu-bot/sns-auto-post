[No.1] 危険警告型

新築アパート買うやつ、ちょっと待ってくれ。

施工原価が1,000万なのに、売値が2,200万。
その差が利益？違う。それが「営業マージン」。

買った瞬間に建物価値は20%吹っ飛ぶ。
表面利回り8%に見えても、10年後に売ると大赤字だったりする。

建築詐欺経験して気づいたんだが、新築は業者都合の商品なんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

不動産会社が絶対に言わないこと。

仲介手数料は「両手仲介」で2倍になる。
売主からも、買主からも、両方から3%取ってるわけ。

その利益のために、わざと情報を囲い込んだり、
成約を長引かせたり、マッチング悪くしたり……

ぶっちゃけ、業者の利益と顧客の利益は逆向きなんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.3] 失敗談型

施工業者から詐欺られた話。

「総額5,000万で新築アパート建築」という契約。
着工から3ヶ月で追加工事の請求が来た。

「土壌改良費で800万」
「構造補強で600万」

完成時には総額6,800万に。契約書には「追加工事特約」という死文が。

弁護士に相談したら「証拠不足」で泣き寝入り。
今は施工契約書のチェックリスト化してる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.4] 初心者罠型

投資初心者が99%やる間違い。

「物件を買う前に、融資枠を調べない」

銀行から融資が出る額と、儲かる物件の値段は別。
融資が出やすい新築は割高。融資が出にくい中古は割安。

結果、融資審査に通るから＝良い物件ではないんだよな。

融資可能額の調査が最初のステップなんだが、ほぼ誰もやらない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.5] 融資・ファイナンス型

融資を断られる人の共通点。

「年収に対して負債比率が高い」

銀行は「返済比率」（年間返済額 ÷ 年収）を見てる。
これが35〜40%を超えるとアウト。

会社員で年収500万なら、年間返済は175万が上限。
それ以上の融資は出ない。

すでに車ローンや住宅ローンがあると、その枠を圧迫する。
現状把握が99%の人で足りてない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.6] 利回りの罠型

「表面利回り10%」と書いてある物件。

年間家賃100万 ÷ 購入価格1,000万 = 10%

でも実質は、空室率5%、管理費10万、修繕費15万、税金12万……

実質利回りは、（100万 − 52万） ÷ 1,000万 = 4.8%

表面利回りと実質利回りは全く別の数字。
業者はいつも表面しか言わない。

自分で逆算する癖をつけたほうがいいな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.7] 市場データ型

2024年、首都圏の新築マンション価格は8,600万円が平均。

去年比で+12%。
5年前と比べると+32%。

一方、中古マンションは微増・横ばいの地域がほとんど。

新築と中古の価格乖離（かいり＝差が広がること）が加速してる。
新築投資の割に合わなさが、データで証明されてる状態。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

不動産って「3つのタイミング」がある。

【買い時】
金利が高い、市場が冷えてる、割安物件が出ている状態。

【保有時】
キャッシュフローを重視。利回り追求。

【売り時】
金利が低い、市場が過熱、買い手が増えてる状態。

今は「金利上昇局面」だから、昨年買った物件も評価下がってる。
5年前に買った物件の売却検討は今がタイミングかもな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.9] 自宅購入型

マイホーム買うときに不動産会社は言わないこと。

「ローンは35年が当たり前」は嘘。

完済が80歳超になる人、ザラにいる。
60歳で定年退職しても、あと20年ローン支払い続けるわけ。

年金時代の住宅ローンは、月々の支払いより「借金の不安」がデカい。

完済時期から逆算して、購入時期と額を決めるべきなんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.10] 管理・運用型

物件買ったら終わりだと思ってる人、多すぎ。

実際は、ここからが本業。

入居者クレーム、修繕対応、原状回復工事、
滞納催促、退去立会、募集活動……

会社員の副業で「丸投げ」すると、
管理会社の搾取＋手抜き対応で、あっという間に利回りが10%→5%になる。

自分でチェックする仕組みが必須。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.11] 相続・贈与型

親が不動産持ってるなら、今のうちに整理しておくべき。

親が亡くなると「相続争い」になりやすい。

特に「複数の兄弟がいる」「不動産が複数ある」場合は、
誰が何を相続するか、揉めるんだよな。

遺言書がないと、話し合いで決まるまで売却もできない。
最悪、相続税は払わせられるのに、現金化できない状態に。

親が元気なうちに「遺言 + 不動産の整理計画」を立てるべき。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.12] エリア選定型

「今後伸びるエリア」を見極める方法。

データで見るのは、駅徒歩圏内で
【新築マンション供給量】【人口流入】【再開発計画】

この3つが揃ったエリアは確実に上昇トレンド。

逆に、ニュータウンで「新築供給ゼロ」「人口減少」「再開発計画ナシ」なら、
10年後は確実に下がってる。

感情で「住みたい」ではなく、データで「買い」を判断するんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.13] 金利・ローン型

住宅ローン「変動金利」と「固定金利」の選択を間違えてる人が多い。

変動金利は、今は0.4%だけど、5年後に2%になる可能性もある。

年間返済額が100万の物件が、120万に跳ね上がるわけ。

会社員で「給与が上がる保証がない」なら、
最初から固定金利で月々の負担を確定させるべき。

安さに釣られて、後で後悔するパターンが9割。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.14] 問いかけ型

「月々のローン返済 < 家賃収入」で儲かってると思ってますか？

実はそれ、まやかし。
管理費、修繕費、空室損、税金、保険……を引いたら赤字かもしれない。

本当の利益は「キャッシュフロー」ベースで計算しないと、
10年後に「え、利益出てなかった」になる。

感覚値で投資判断してないか、一度チェックしたほうがいい。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.15] 法改正・制度型

2024年の税制改正で「相続税」の計算ルールが変わった。

不動産の「小規模宅地等特例」の適用範囲が縮小。

親から相続する土地が、100%評価減だったのが、
今は70%に。

数千万円の相続税が急に上がる家族が多発中。

「相続対策」を親の世代でやるなら、この改正後がチャンス。
早めに税理士に相談したほうが、子世代の負担が全然違う。
