[No.1] 危険警告型

「利回り12%以上」って聞いたら即逃げろ。

利回りが高い＝リスク高い。
新築木造アパート・サブリース物件・
不動産会社が売主の場合、特に危ない。

建築詐欺で1,000万失った俺だから言える。
美しい数字の裏には必ず落とし穴がある。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

不動産会社が物件を売りたい時、
必ず「今が買い時」と言う。

本当は「今が売り時」なんだよな。
つまり売り主が出した物件=売り主にとって
手放すメリットがある時期。

その価値をお前は見極められるか。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.3] 失敗談型

屋根・配管で300万。
そっから毎年100万の出費。

利回り計算に修繕費を入れてなかった。
数字だけで買うと死ぬ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.4] 初心者罠型

投資初心者が買う物件の共通点：

・駅から徒歩10分以上
・築30年以上
・利回り8%以上

この3つ揃ってたら、
不動産会社に値引き要求されてる物件。

つまり「誰も欲しくない物件」だったりする。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.5] 融資・ファイナンス型

融資が通らない人の共通点：

・頭金が全資産の50%以上
・勤続3年未満
・負債比率60%超
・自営業で経営を3期以上見られてない

銀行は「返せる人」じゃなく
「継続的に稼げるか」を見てる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.6] 利回りの罠型

表面利回り10%＝
年間賃料 / 購入価格

実質利回り6%＝
（賃料 - 経費） / 購入価格

経費＝管理費・税金・修繕積立・保険・空室率等

見るべきは実質利回り。
表面で判断する奴は破産する。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

値上がり益（キャピタルゲイン）狙いは

30年後、家の価値はほぼゼロ。
それでも買うなら、覚悟を決めて買え。

[No.12] エリア選定型

東京の不動産が上がってる理由：

・オリンピック後も企業本社が集中
・外国人が買う（資産逃がし）
・地方の人口が流入
・都心部の駅直結案件が少ない

つまり「限定性」と「流動性」が高い。

地方物件を買う時は
この2点が本当にあるか確認しろ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.13] 金利・ローン型

住宅ローン2.0% vs 投資ローン3.5%

その差1.5%＝年間150万円の負担増。

投資物件は自宅より
金利が高い理由は簡単。

銀行は「居住」を優遇する。
投資は「損失リスク」として見てる。
