[No.1] 衝撃ファクト型

会社員の平均有給消化率って日本で50%弱なんだよな。
つまり年10日近く、給与確定してるのに使わずに消える。

ドブに捨ててるのと変わらねえ。

その10日あれば副業始められるし、
不動産の内見も回れるし、
飲食店の下見だって全然余裕。

キャリア教が「有給は気がひける」って洗脳してるだけ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.2] 逆張り意見型

「頑張ることが人生を変える」
って言葉、嘘だと気づいた。

正確には「頑張る場所を選べる人が人生を変える」んだ。

会社内での頑張りは、会社の利益にしかならん。
自分の事業での頑張りは、自分の資産になる。

同じ頑張るなら、後者選ばねえ理由がねえよな。

推奨投稿時間：朝8時 / 夜20時

[No.3] 体験談型

俺が営業成績1位のときより、
今月の不動産投資の利回りが出たときの方が
心の安定感ハンパねえ。

なぜなら、1位の成績は上司に左右されるけど、
資産からのリターンは自分でコントロール可能だから。

同じ時間使うなら、
他人の評価より自分の資産増やした方がまじで楽。

推奨投稿時間：朝7時 / 昼12時 / 夜19時

[No.4] 行動促進型

明日から1つだけやること。

「今月の有給残数を数える」

たったこれだけ。
だけど、これが全ての始まり。

自分がどれだけ給与確定の時間を持ってるか知らない奴が、
副業やキャリア変更を語っても机上の空論。

まずは数字を見ろ。

推奨投稿時間：夜20時

[No.5] 比較対照型

【以前の俺】
→毎日会社へ。帰宅して副業で疲弊。
→有給は「申し訳ない」で取らず。
→5年後も同じ給与。資産ゼロ。

【今の俺】
→会社は週4日。残り1日は副業・投資。
→有給は戦略的に物件確認に充てる。
→3年で不動産3棟。毎月の利益20万超え。

変わったのは、時間の使い方と「ルール破り」の覚悟だけ。

推奨投稿時間：朝8時 / 昼14時

本当の仕事は副業と投資だから。

立場は会社員。実質はノンキャリア。
この両立が最強。

[No.9] 失敗談型

新卒から5年間、会社の出世に全力。
資格も取った。評価も上げた。

その代わり、副業0。資産0。

35歳の今、同期の出世競争から外れてわかった。
会社員のキャリアは、定年まで続く保証はない。

失った5年と時間は戻らねえけど、
今からなら遅くねえ。

その時に気づいてほしいから、
俺は今この話をしてる。

[No.10] 初心者向け解説型

「ノンキャリア」って言葉、俺の造語じゃないけど、
わかりにくいから定義する。

【ノンキャリアとは】
→立場：会社員（給与・福利厚生あり）
→本業：副業・投資・個人事業
→時間配分：会社2-3割、本業7-8割
→目的：5年で会社給与を不要にする

給与は欲しい。でも人生は自分のもの。
この両立を目指す働き方。

推奨投稿時間：朝9時 / 昼13時

[No.11] 上級者向け洞察型

有給・休職・副業を戦略的に使いこなす順序。

1. 有給を「疲労回復」じゃなく「事業時間」に充てる
2. 副業で月5万の利益を作る（会社給与の心理的依存を減らす）
3. 休職で脳をリセット＆スケーリングを考える
4. 会社勤務を段階的に減らし、時間を事業に振る

多くの奴は順序を逆にしてる。
だから失敗する。

推奨投稿時間：朝8時 / 昼14時 / 夜22時

[No.12] 長期思考型

5年後に年収1000万。
それ、会社員で目指してないか？

俺は違う視点で考えてた。

5年で、給与に頼らない月50万。
これ達成したら、会社員であることが選択肢になる。
強制じゃなくてね。

その時点での自由度、ハンパねえよ。

10年でそれを100万にする。
会社員やめるかやめないか、そん時に決める。
