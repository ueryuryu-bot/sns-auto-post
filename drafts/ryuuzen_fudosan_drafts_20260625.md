[No.1] 危険警告型

新築木造アパート、まじで危ない。

建築詐欺経験した俺が言うのは、
「新築時点での価値 = 完成時点で30%目減り」
が相場だったりする。

にもかかわらず
フルローンで買わされる投資家が後を絶たない。

売却時に大赤字。
銀行も「売却損に応じません」と冷徹だ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

不動産会社が言わないこと。

それは「この物件、
利益乗せすぎてます」って話だ。

手数料3%だけじゃ足りない。
物件仕入れ値と売却価格の間に
200万〜500万の利益を忍ばせるのが業界標準。

買う側はそれを「相場」と思い込む。

んだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.3] 失敗談型

俺が2011年に買った
木造アパート（築30年）の話。

表面利回り12%に釣られた。
実際は空室率25%、
修繕費年200万。

実質利回りは3%だった。

当時の俺は「数字」しか見てなかったんだよ。
物件を見に行くのは最後だった。

今はその逆をやってる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.4] 初心者罠型

投資初心者がやってしまう失敗。

それは「自分で管理しよう」という罠だ。

サラリーマン投資家が
物件の修繕・クレーム対応・入居者管理
を自分でやると、

本業がおろそかになり、
給料査定でマイナスを喰らう。

管理費3〜5%の出費は
「必要経費」だったりする。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.5] 融資・ファイナンス型

融資を断られる人の共通点。

勤続年数3年未満、
転職が多い、副業の赤字申告。

銀行は「安定性」を見てる。
年収じゃなく「継続性」だ。

あと意外と多いのが
「融資の事前審査を通してから物件を探す」
をやらないパターン。

スコア算出してから探せ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.6] 利回りの罠型

表面利回り（年間家賃÷物件価格）
に騙されてはいけない。

実質利回りは
（年間家賃−年間経費）÷物件価格

経費には
管理費・固定資産税・保険・修繕費・空室ロス。

表面12%が
実質4%になることなんて珍しくない。

物件を見るなら
必ず逆算から入れ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.7] 市場データ型

2024年度のデータから見えるもの。

首都圏の新築マンション価格、
過去最高を更新中。

一方で地方の築古木造アパートは
利回り10%超えが当たり前。

でも高利回り = 高リスク。

買値と売値が離れてる地域は
出口戦略がクッソ難しい。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

今が売り時か買い時か。

簡潔に言うなら
「金利が上がってる局面では売り時」だ。

なぜなら買い手の借入コストが跳ね上がるから。
買値が下がる。

逆に金利が下がってく局面では
買い時になることが多い。

ただし底値は誰にも分からない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.9] 自宅購入型

家を買うときに誰も教えてくれないこと。

それは「買った瞬間、負債になる」ってことだ。

新築戸建てはまじで5年で
1000万円ダウンする。

ローン残債と売却価格の逆転は
珍しくない。

自宅は「資産」じゃなく「消費財」。
その割り切りがないと後悔する。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.10] 管理・運用型

物件を持ってから始まる本当の苦労。

それは修繕との付き合い方だ。

屋根・外壁・配管は
必ず予想より早く傷む。

年1回の修繕積立金だけでは足りず、
急な大修繕で100万単位の出費。

融資を組んでるときの修繕は
キャッシュフローを直撃する。

予備資金は物件価格の5%。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.11] 相続・贈与型

不動産×相続で揉める家族のパターン。

親が複数の物件を子供に残すと、
必ず「売却するか、保有するか」で割れる。

その時点で相続税の評価額と
実売却価格に差が生まれて、
兄弟で損得が変わる。

相続は「不動産」じゃなく「現金化」を想定して
遺言を組むべきだったりする。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.12] エリア選定型

なぜそのエリアは上がるのか。

駅徒歩10分以内、
かつ「周辺に競合物件が少ない」地域。

逆に下がるのは
駅から遠いくせに
類似物件が100軒以上ある地域だ。

需給バランスが全てを決める。

データベースで競合数を確認しないで
物件を買うのは危険。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.13] 金利・ローン型

住宅ローンと投資ローンは別モノ。

住宅ローンは
金利2%前後・35年返済。

投資ローンは
金利3.5%〜5%・20年返済。

同じ融資額でも月々の返済額は大きく違う。
キャッシュフローが赤になるケースも多い。

投資ローンは表面利回りで選ぶな。
実質利回り - 金利 = 年間利益 で考えろ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.14] 問いかけ型

「不動産は絶対に値上がりする」
って思い込みませんか？

人口減少局面で
値上がりする不動産は、
実は「限られた場所」だったりする。

全ての不動産が資産になるわけじゃない。

そこをリアルに見つめてから
買う決定をすべき。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.15] 法改正・制度型

2023年の相続登記義務化。

知らないと損する人多い。

相続後、登記せずに放置すると
罰金（過料）が課される。
最大10万円。

さらに売却時に
相続登記の手続きが煩雑化して
費用が嵩む。

親が亡くなったら
速やかに登記を完了させること。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか
