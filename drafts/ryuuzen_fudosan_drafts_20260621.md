[No.1] 危険警告型

「新築木造アパート」で満室なのに
赤字になる投資家、まじで多い。

理由は建築会社の利益上乗せ。
1億2000万で建てた物件を
1億5000万で売られてる。

その時点で利回りは死んでる。

新築は「売上」じゃなく
「建築原価」を知らないと
最初から負けてんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

---

[No.2] 業界本音型

不動産会社は
「売ったら終わり」なんだよ。

あいつらのKPI（業績評価）は
成約件数と売上高。

その後、お前の物件が
赤字になろうが空室になろうが
1円も損しない。

だから利回りも根拠も曖昧なまま
「いい物件ですよ」で押し通す。

投資家の味方じゃなくて
販売手数料の味方ってわけ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.3] 失敗談型

融資10億超えする前に
1棟1000万の木造アパートで
建築詐欺に引っかかった。

施工会社が倒産して
追加工事費400万が
突然請求されてきた。

当時は知識も弁護士も持たなくて
そのまま払うしかなかった。

今はそういう案件は
契約前に全て潰せる。
経験が一番高い授業料だったりする。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.4] 初心者罠型

不動産投資を始める人が
必ずやる間違い。

「表面利回り8%」の物件を見て
「いけそう」って判断してる。

実質利回りは
（年間家賃 − 税金・保険・修繕費・管理費）÷
購入価格 × 100

大体3〜4%まで落ちる。

表面だけで判断する奴は
2年で気づく。あ、これ赤字だ。って。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.5] 融資・ファイナンス型

銀行が融資を断る人の
共通点は2つ。

①頭金が少ない
②返済比率が高い
（年間返済額 ÷ 年間収入）

銀行は年収600万でも
返済比率が40%超えなら落とす。

逆に年収400万でも
返済比率25%以下なら通る。

数字の銀行員には
「稼いでる感」より
「返せる構造」を見せろ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.6] 利回りの罠型

「表面利回り10%」という魔法の言葉。

でもな、建物は毎年劣化する。
5年後は修繕費で100万単位。

家賃も下がる。競合増えたら3年で−5%。

表面利回り10%の物件は
実質5年目には3%になってる。

それでも買うなら
「資産防衛のための物件」って
割り切らないと地獄だぞ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.7] 市場データ型

2024年の投資用マンション相場、
東京オリンピック後から
+15〜20%上昇してる。

でも空室率も同時に上昇中。
首都圏の空室率は過去最高の15%超え。

つまり「高値掴みしやすい時期」
なんだよ。

数字が上がってるから
心理的に買いたくなるが
需給バランスは既に悪い。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

「今が売り時か買い時か」の判断、
シンプルだ。

①金利が上昇局面 → 売り時
（物件価格が下がる前に手放す）

②金利が低下局面 → 買い時
（今後10年単位で上昇益の可能性）

今は日銀が利上げを示唆してる。
つまり、ここからは
買い増しより整理の時期。

周りが買ってても関係ない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.9] 自宅購入型

家を買うときに
不動産屋は言わないこと。

購入価格の3〜5%が
仲介手数料・登記料・税金で消える。

3000万の家なら100万以上が
手数料で蒸発する。

さらに10年住むと
修繕費200万前後必要。

「買ったら終わり」じゃなくて
「買ってから金がかかる」
これが現実。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.10] 管理・運用型

投資物件を買ったら
本当の仕事が始まるんだよな。

家賃滞納者への対応。
突然の配管破裂。
クレーマー入居者。

月1回の巡回。年2回の修繕計画。
確定申告の帳簿整理。

利回り計算では出ない
「手間賃」がのしかかる。

だから管理会社費用は
「経費」じゃなくて
「時給換算で見なきゃ」ダメなんだ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

「ここなら建て替えして利益出る」

[No.13] 金利・ローン型

投資ローンと住宅ローン、
金利差は0.5〜1.5%。

100万借りるのと1億借りるのなら
この差は年300万にもなる。

でも銀行は
「35年固定が安心です」
って言葉で
後発のリスクを隠す。

金利上昇局面では
10年固定で4回借換えするのと
35年固定を一度で決めるのとで
最終返済額が2000万違う。

金利は「選ぶ権利」だ。
