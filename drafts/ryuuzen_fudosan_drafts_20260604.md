[No.1] 危険警告型

新築木造アパート。
表面利回り10%。

まじでやめたほうがいい。

理由は木造の耐用年数（税務上22年）と
実際の融資期間のズレ。
25年ローンなら残り3年は
返済しながら価値ゼロの負債を抱える。

新築プレミアムで買値600万のが
3年で300万の価値しかない物件、
いっぱい見てきた。

[No.2] 業界本音型

不動産営業が「今が買い時」と言うのは
本当は「今が売り時」の意味。

彼らの給料は売上手数料だから
市場がいつだろうと
「今買いましょう」としか言わない。

金融機関の融資審査が甘い時期＝
破綻者が増える時期。

営業の笑顔と市場の危険度は
反比例する。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.3] 失敗談型

融資額10億超の中で
一番痛い失敗は建築詐欺。

コンテナハウスを「新工法」と言われて
2000万投資したら
実は既存技術の寄せ集めで
5年で雨漏り・遮熱性ゼロ。

建築会社は破産。

その時学んだのは
「新しい＝儲かる」ではなく
「実績ある＝安全」が正解ってこと。

[No.4] 初心者罠型

不動産初心者が必ずハマる罠：
「利回り」で物件を選ぶ。

実は大事なのは
・立地の需要（空室率）
・金利と返済比率の相性
・修繕費の現実性

利回り12%と利回り6%の物件なら
確実に6%の方が儲かる場合、
マジであります。

数字だけ見るから失敗する。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.5] 融資・ファイナンス型

融資を断られる人の共通点。

それは「返済比率」を理解してない。

年収500万で既存ローン年100万あれば
新規融資は年利3%で
最大約1200万までが限度（返済比率35%基準）。

多くの人は年収だけ見て
「自分は2000万借りれる」と勘違いしてる。

金融機関は年収÷（既存返済+新規返済）を見てる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.6] 利回りの罠型

表面利回り8%の物件を買った。

実質利回りは4.2%だった。

理由：
・管理費：年間収入の8%
・空室ロス：想定以上に2ヶ月/年
・修繕費：毎年100万超

「あれ、実質利回り」と気付く前に
既に15年ローン組んでる人、多い。

表面と実質の差を
最初に計算しない奴は
投資家じゃなくてギャンブラー。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

新しい物件は需要があるうちに買う流れ。

投資判断は「今いくら」ではなく
「5年後いくら」で考えるべき。

[No.8] 売却タイミング型

売却タイミングの判断軸。

「利益が出たから売る」は素人。

正解は：
・金利上昇局面（今ここ）→売却有利
・新築供給増加地域→値下げ圧力発生前に売却
・融資期間残り5年以下→次の買い手が融資困難

逆に買い時は金利底打ち前。

市場ではなく金融機関の貸し出し姿勢を見る。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.9] 自宅購入型

持ち家を買う人へ。

誰も言わないけど
都市部の一軒家は
実は資産ではなく負債の可能性が高い。

理由：
・土地価値より建物の老朽化速度が早い
・リノベーション費用が想定以上
・売却時に買い手がつきにくい

年収に対してローン返済比率30%以上なら
買わない方が15年後の資産は増える。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.10] 管理・運用型

物件買ったら終わり、は大間違い。

実際の苦労：
・家賃滞納時の対応（回収コスト）
・退去者の原状回復トラブル
・修繕の見積もり取得（悪徳業者避け）
・空室期間の税負担（ローンは止まらない）

「管理会社に任せてたら楽」って思ってる人、
実際は管理費+手数料で利回り2%削られてる。

運用＝経営。受け身は絶対ダメ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

[No.11] 相続・贈与型

不動産と相続は家族を壊す。

親が複数の物件を子に残すと
「売ろう派」と「残したい派」で必ず揉める。

相続税対策で不動産を買わされた家族の
9割は後悔してる。

理由：
・売却しても買い手つかない古い物件
・兄弟で共有すると身動き取れない
・相続税より維持費の方が高くつく

生前に「売却」前提で整理する親が正解。

[No.12] エリア選定型

エリアで全て決まる。

クソ多い。

融資の損得は
表面金利ではなく
「税務効果を含めた実質金利」で判断しろ。
