[No.1] 危険警告型

「利回り12%の物件、買ったら地獄です」

新築木造アパート、表面利回り12%。
でも実は5年目には大規模修繕で300万飛ぶ。
木造は劣化速度が鉄骨の2倍なんだよな。

利回りだけ見て買う奴、
マジで後悔する。
残債と修繕費のダブルパンチで
売却も身動き取れない。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.2] 業界本音型

「不動産会社が『この物件はやめた方が』と言わない理由」

手数料だから。

駅から遠い、人口減少地、
築40年…全部知ってる。
でも仲介手数料は売上。

投資家が損しようが、
5年で空室化しようが
業者のメリットじゃない。

儲かる物件と売れる物件は別なんだよな。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.3] 失敗談型

「建築詐欺で800万失った話」

新築物件、施工会社がいきなり夜逃げ。
基礎が規格外、壁も傾いてた。

完成保証保険入ってたのに
「瑕疵ではなく設計ミス」で却下。
弁護士費用150万かけて
やっと650万の和解。

150万失ったら、
年間利回りが消し飛ぶんだよ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.4] 初心者罠型

「投資初心者が『表面利回り』だけで物件選ぶ罠」

表面利回り＝年間家賃÷物件価格。
カンタンだから素人は信じちゃう。

でも実質（ホント）の利回りは
税金、保険、修繕費、空室リスク引いた後。
そっちが7%、実はマイナスなんてザラ。

計算面倒だからこそ、
ここが勝負分かれ目。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.5] 融資・ファイナンス型

「融資が通らない人、共通点3つあるんだよな」

①年収の30倍超の融資希望
②3年以内に転職
③属性は良いのに個人信用情報が汚い

金融機関は返済能力より
『返す気があるか』の信用を見てる。

属性ガチガチでも
信用情報ブラックなら落ちる。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.6] 利回りの罠型

「年間家賃600万、物件価格5000万。利回り12%？違います」

【表面利回り】
600万÷5000万＝12%

【実質利回り（現実）】
（600万−空室50万−税金100万−修繕30万−保険20万）÷5000万＝8.8%

4ポイント差。
年間40万の違いなんだよ。

長期で見たら、致命的。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.7] 市場データ型

「首都圏築20年RCマンション、去年比で平均−8%。買い時？売り時？」

答え：売り時。

金利上昇局面で
築古は値下げ幅が大きくなる法則。
2年後さらに−5%来る確度高い。

手持ち物件あるなら、今が出口。
買い増しは様子見推奨。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.8] 売却タイミング型

「売り時の判定『1つの指標』」

自分の保有コスト（年間支出）÷売却益の年間還元率＞5年

超えたら売ろう。
時間価値が逆転してる。

あとは感情。
「もう一年持ちたい」ならボロ負け。

プロ投資家は
感情と数字を分ける。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.9] 自宅購入型

「マイホーム購入、営業は言わないけど実は『35年ローンは罠』なんだよな」

銀行が儲かるから、長期を勧める。
でも金利上昇時代に長期固定は消える。

変動から固定に切り替える時、
プラス1.5%とか普通。
毎月返済＋5万。

自宅こそ『繰り上げ返済の覚悟』が必須。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.10] 管理・運用型

「物件買ってからが本当の苦労。管理会社選び間違えたら年間100万損するぞ」

賃貸管理手数料：家賃の5〜8%
それ以上取る業者＝詐欺に近い。

でも安い業者は対応ゼロ。
クレーム放置、修繕ぼったくり。

『安かろう悪かろう』が不動産では命取り。
中価格帯で信頼できる業者を選ぶ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.11] 相続・贈与型

「不動産×相続で兄弟絶縁、マジで多いんだよな」

兄弟で分割できない不動産が揉める。
売却しようにも全員同意必須。

相続する前に『誰がいくらもらうか』を
紙に残すだけで、家族が潰れないぞ。

[No.13] 金利・ローン型

「変動金利0.5%、固定金利2.0%。迷ったら？」

迷ったら固定一択。
金利上昇局面だから。

変動は『ギャンブル』。
銀行も5年後の金利なんて予測できない。

固定なら『見える敵』。
計算が立つ。
投資は見える敵でやるんだよ。

[No.14] 問いかけ型

「あなたの『現在の家賃』と『買ったら毎月の返済額』同じですか？」

同じなら買うな。
家賃≧ローンは、資産にならない。

買うなら家賃より返済が低い物件。
その差が年間の『資産形成額』になる。

この基本守らない奴、15年経ってもゼロ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時のいずれか

[No.15] 法改正・制度型

「2024年から不動産譲渡所得の『長期保有特例』が変わったの知ってた？」

15年超保有で14.21%
→20年超で『20.315%の優遇』
（=通常39.63%から大幅減税）

つまり『15年で売るな』が
『20年持つ』にシフト。

早期売却は税制的に損してる。出口戦略は20年単位で。
