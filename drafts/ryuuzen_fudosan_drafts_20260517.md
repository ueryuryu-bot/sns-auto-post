**[No.1] 危険警告型**

新築木造アパートだけは買うな。
施工不良で10年で躯体が傷むんだよな。

俺も1棟目で初期不良に気づかず
オーナーチェンジで買わされた。
修繕費だけで年間300万超。

新築は銀行も融資しやすいから
「売る側も買わせたい」んだ。

見た目より構造。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

**[No.2] 業界本音型**

不動産会社は「物件の欠点」を
ぜっタイに最初は言わない。

仲介手数料は売上だから
「この立地は駅から遠い」「築古で融資が難しい」
こういう話は後出し。

業界の人間だからこそ言えるが
信頼できる会社の見分けは
「最初に悪いことを言うか」だ。

推奨投稿時間：朝10時 / 昼14時 / 夜20時

**[No.3] 失敗談型**

建築詐欺で1,200万失った時の話。

施工会社が「手抜き工事 → 見積以上に請求」
やられたんだよ。当時は知識不足で
契約内容を甘く見ていた。

今思うと、複数の独立した設計士に
チェックさせるべきだった。

損失は学費だと割り切った。

推奨投稿時間：朝11時 / 夜19時 / 夜22時

**[No.4] 初心者罠型**

「利回り10%の物件が出ました」は
ほぼ罠だと思え。

なぜなら空室率・管理費・修繕費
を全く含まない「表面利回り」だから。

実際の手残りは5%以下だったりする。

初心者は数字の大きさで判断しがちだが
細かい経費を全部引いた「実質利回り」で
比較しなきゃ失敗する。

推奨投稿時間：朝9時 / 昼12時 / 夜21時

**[No.5] 融資・ファイナンス型**

融資を断られる人の99%は
「自分の信用スコア」を知らない。

年収・勤続年数・借金額・支払い遅延歴
これらを銀行はスコアリング（=数値化）する。

金融機関の本音は
「高い年収より、返し続けてくれるか」だ。

会社員なら給与明細2年分を
今から整えておくべき。

推奨投稿時間：朝8時 / 昼15時 / 夜20時

**[No.6] 利回りの罠型**

表面利回り8% = 実質利回り3.2%

こんなの珍しくねえんだよな。

【見落としがちな経費】
・空室率（実績で2割）
・管理費（毎月2〜5万）
・修繕費（年間2〜5%）
・税金（所得税・住民税）

全部引いたら、銀行金利と大差ない。

不動産は「見た目の数字」じゃなく
手元に残るキャッシュで判断しろ。

推奨投稿時間：朝10時 / 昼13時 / 夜19時

**[No.7] 市場データ型**

2025年の地方物件相場、
マジで下げ止まらないんだよ。

地方：築10年で▲22%

特に「駅から徒歩15分以上」は
売値が希望値の60%まで下がってる。

買う側は「利回りの良さ」に釣られるが
売る時に痛い目を見る。

推奨投稿時間：朝9時 / 昼11時 / 夜21時

**[No.8] 売却タイミング型**

「今が売り時か」の判断は
自分の人生軸で決めろ。

よくある失敗は「金利が上がったから売ろう」
これは後付けの理由。

本当の売却タイミングは
・次の投資資金が必要
・管理の負担を減らしたい
・人生のステージが変わった

相場は二の次。自分の現金フローと
人生計画で判断すべき。

推奨投稿時間：朝10時 / 昼14時 / 夜20時

**[No.9] 自宅購入型**

自宅購入のタイミングで
多くの人は「今が買い時」と思ってる。

違う。本当は「売り時」で判断すべき。

買った時より売れる保証がなければ
その物件は「資産」じゃなく「負債」だ。

特に地方・郊外の一戸建ては
20年後、売値がゼロに近い。

自宅も「いずれ売却する資産」と考えて
買わないと後悔する。

推奨投稿時間：朝9時 / 昼12時 / 夜22時

**[No.10] 管理・運用型**

物件を買った瞬間が
不動産投資の終わりじゃなくて、
むしろ始まりなんだよな。

【実際の手間】
・入居者対応（苦情・クレーム）
・修繕調整（時間取られる）
・税務申告（結構複雑）
・空室期間の管理

サラリーマンで「副業です」とか言う人ほど
この運用負荷を甘く見てる。

管理会社に任せると費用がかさむ
自分でやると時間を取られる

これが不動産投資のリアルだ。

推奨投稿時間：朝9時 / 昼13時 / 夜21時

**[No.11] 相続・贈与型**

親の不動産が相続で揉めるパターンは
ほぼ同じ。

「誰が住むのか」「売るのか」が決まってなくて
兄弟で意見が分かれる。

さらに相続税。評価額の計算間違いで
後から追徴課税されたりする。

遺言書がなかったら最悪。
不動産は現金より相続トラブルが深い。

生きてる間に「この物件の扱い」を
決めておくべき。

推奨投稿時間：朝10時 / 昼15時 / 夜20時

**[No.12] エリア選定型**

「駅近だから価値がある」は
もう古い思考だ。

今は「そのエリアの人口増加率」と
「テレワーク需要」で決まってるんだよな。

郊外でも「駅から遠くても
快適な街づくり」がされてるエリアは上がる。

逆に駅3分でも「人口減＋高齢化率高い」
ならジリ貧。

立地の価値は「駅距離」じゃなく
「そこに人が集まるか」で判断しろ。

推奨投稿時間：朝9時 / 昼11時 / 夜21時

**[No.13] 金利・ローン型**

住宅ローンと投資ローンは
金利が全然違うんだよ。

住宅ローン：年1.2〜1.8%
投資ローン：年2.5〜4.5%

この差で30年返すと数千万変わる。

さらに初心者は「変動金利」で借りて
後から上昇時に焦る。

借りる時は必ず
「固定金利」「返済シミュレーション」
「他行との比較」の3つを実行しろ。

推奨投稿時間：朝8時 / 昼13時 / 夜19時
